Mallas Poligonales

Estructuras de datos para la representación discreta de superficies 3D

Concepto de Malla

Una malla es una colección de vértices, aristas y caras que definen la forma de un objeto poliedral. Es el lenguaje base que las tarjetas gráficas (GPU) entienden para dibujar objetos.

VérticePunto en el espacio (X, Y, Z)
AristaLínea que une dos vértices
CaraSuperficie (Triángulos o Quads)

Estructuras de Datos

Para procesar mallas de millones de polígonos, se usan estructuras como:

  • Half-Edge: Ideal para editar mallas dinámicamente.
  • Face-Vertex: Una lista simple de caras que referencian vértices, común en archivos .OBJ.
