Fundamentos Matemáticos

A diferencia de las funciones tradicionales y = f(x), las curvas paramétricas definen las coordenadas como funciones de una variable independiente t. Esto permite que una curva regrese sobre sí misma o forme lazos.

P(t) = ( x(t), y(t), z(t) )
t ∈

En computación gráfica, esto es vital para la continuidad. Al variar t suavemente, obtenemos una trayectoria fluida, ideal para fuentes de texto (tipografía) y diseño industrial.

Aplicaciones Clave

  • NURBS: El estándar en ingeniería (CAD) para representar círculos y elipses de forma exacta.
  • Interpolación: Para suavizar el movimiento de cámaras en videojuegos.
  • Fuentes: Formatos como .OTF o .TTF usan estas curvas para escalar letras sin perder calidad.
