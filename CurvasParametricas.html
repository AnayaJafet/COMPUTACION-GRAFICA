Concepto y Fundamentación

Una curva paramétrica es una representación donde las coordenadas de los puntos no se definen mediante una relación directa entre ellas, sino como funciones de una variable independiente común denominada parámetro (t). En el espacio 3D, esto se expresa como P(t) = (x(t), y(t), z(t)).

Propiedades

Permiten representar formas que se auto-intersecan o se cierran, algo imposible para funciones explícitas estándar. Ofrecen una resolución infinita.

Continuidad

Se estudian niveles de suavidad: C0 (unión simple), C1 (tangentes coincidentes) y C2 (curvatura constante), vital para reflejos en superficies.
